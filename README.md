# 🌸 KuroSakura

A personal anime streaming site — built for fun, learning, and watching anime without the clutter of typical sites.

## ✨ Features

- 🎬 Stream anime episodes with sub/dub options
- 📚 Browse trending and popular titles
- 🔖 Save/bookmark anime to watch later
- ✅ Track watched episodes
- 🔍 Search episodes within a series
- 📱 Responsive design — works on mobile and desktop
- 🎨 Custom dark theme with a sakura-pink aesthetic

## 🔗 Live Site

👉 [kurosakura.vercel.app](https://kurosakura.vercel.app)

## 🛠️ Built With

- HTML, CSS, JavaScript (vanilla, no frameworks)
- [AniList API](https://anilist.co/) for anime metadata
- Cloudflare Workers for backend proxying
- Hosted on Vercel + Cloudflare

## 📌 Status

Actively in development — new features and fixes rolling out regularly.
